We are thrilled to announce the release of ThingsBoard version 3.7.0! This update brings numerous features designed to enhance user workflows and boost performance. Highlights include the Migration to JDK 17, enriched localization support, upgraded data visualization capabilities, enhancements to existing widgets, and the addition of several new ones.<\/p>\n\n\n\n

Higher performance and better overall user experience<\/h4>\n\n\n\n

In this release, our main aim is to make your experience smoother and safer. We’ve worked on a bunch of behind-the-scenes improvements to make the software run faster, keep your data more secure, and ensure it’s more reliable than ever. Here’s a look at what’s changed:<\/p>\n\n\n\n

    \n
  1. Migration to the Latest Java: We’ve upgraded from JDK 11 to JDK 17, bringing in new features and improvements that help the software run more efficiently and securely.<\/li>\n\n\n\n
  2. Spring Framework Update: We’ve updated from Spring version 5.3.26 to 6.1.8. This update introduces new features and enhancements, enhancing your experience while using the software.<\/li>\n\n\n\n
  3. Docker Image Update: Our Docker images have been updated from bullseye-slim to bookworm-slim. This update includes bug fixes and new features, ensuring a smoother and more secure deployment environment.<\/li>\n\n\n\n
  4. Faster Data Access: We’ve tweaked how the software talks to the database, so retrieving information happens in a flash, giving you the answers you need in record time.<\/li>\n\n\n\n
  5. Smoother Operations: We’ve added a new Housekeeping service to take care of those long, complicated tasks in the background, so you can focus on what matters most.<\/li>\n\n\n\n
  6. Improved Security: Your safety online is crucial, so we’ve strengthened the software’s defenses to keep your data even more secure.<\/li>\n\n\n\n
  7. Reduced Memory Footprint: By optimizing how the software uses memory, it now runs more efficiently, even on devices with limited resources.<\/li>\n\n\n\n
  8. Increased Database Throughput: We’ve improved how the software handles lots of people using it at once, so it stays fast and responsive no matter how busy things get.<\/li>\n<\/ol>\n\n\n\n

    Enhanced  Localization Support<\/h4>\n\n\n\n

    Before release, to make changes to an existing language, you had to download a file with the current set of terms in the chosen language, make your modifications, and then send the updated file to our support for them to implement the changes. Adding a new language was not supported directly on the platform and required similar intervention from support.<\/p>\n\n\n\n

    With the new update, localization support has been significantly enhanced. You can now easily add languages that were not originally available on the platform. The improved localization support feature allows you to search by languages and term keys, filter by unfilled, modified, or new keys, and track translation progress more effectively. We’ve introduced a new, simpler format called Basic, making the translation process more user-friendly. The ability to edit translations in the Advanced tab for a specific language, using the JSON format, remains available for those who prefer this method.<\/p>\n\n\n\n
